# 东方财富数据API接口文档

> 获取于 https://f0ecde19.pinme.dev/ 

## 1. 股票实时行情接口

### 接口名称
获取股票实时行情数据

### 请求地址
```
GET https://push2.eastmoney.com/api/qt/ulist.np/get
```

### 请求参数
| 参数名 | 必选 | 类型 | 说明 |
|--------|------|------|------|
| fltt | 是 | int | 固定值2，表示返回浮动行情 |
| fields | 是 | string | 指定返回字段，多个字段用逗号分隔 |
| secids | 是 | string | 证券代码列表，多个用逗号分隔，格式：市场代码.证券代码 |
| _ | 否 | long | 时间戳，防止缓存 |

### 字段说明
- f2: 最新价
- f3: 涨跌幅（百分比）
- f4: 涨跌额
- f6: 成交额
- f7: 振幅
- f8: 换手率
- f104: 上涨家数
- f105: 下跌家数
- f106: 平盘家数
- f12: 代码
- f13: 市场标识（0=深市，1=沪市）
- f14: 名称

### 响应示例
```json
{
  "rc": 0,
  "rt": 11,
  "svr": 175644146,
  "lt": 1,
  "full": 1,
  "dlmkts": "",
  "data": {
    "total": 5,
    "diff": [
      {
        "f2": 3975.1,
        "f3": -0.28,
        "f4": -11.12,
        "f12": "000001",
        "f13": 1,
        "f14": "上证指数"
      }
    ]
  }
}
```

## 7. A股个股排名接口（资金流向/涨跌幅等）

### 接口名称
获取A股个股排名列表（支持按净流入、涨跌幅等字段排序）

### 请求地址
```
GET https://push2.eastmoney.com/api/qt/clist/get
```

### 请求参数
| 参数名 | 必选 | 类型 | 说明 |
|--------|------|------|------|
| pn | 是 | int | 页码，从1开始 |
| pz | 是 | int | 每页数量，最大200 |
| po | 是 | int | 排序方向：1=降序，0=升序 |
| np | 是 | int | 固定值1 |
| fltt | 是 | int | 固定值2，表示返回浮动行情 |
| invt | 是 | int | 固定值2 |
| fid | 是 | string | 排序字段，如 f62(主力净流入)、f3(涨跌幅)、f6(成交额) |
| fs | 是 | string | 市场筛选条件，多个用逗号分隔 |
| fields | 是 | string | 返回字段，多个用逗号分隔 |

### 市场筛选条件（fs参数）
| 值 | 说明 |
|----|------|
| m:0+t:6 | 深圳A股 |
| m:0+t:80 | 创业板 |
| m:1+t:2 | 上海A股 |
| m:1+t:23 | 科创板 |

常用组合：`m:0+t:6,m:0+t:80,m:1+t:2,m:1+t:23`（全部A股，排除ETF/债券/基金）

### 常用字段说明
| 字段 | 说明 |
|------|------|
| f2 | 最新价 |
| f3 | 涨跌幅（百分比） |
| f8 | 换手率 |
| f12 | 代码（纯数字） |
| f13 | 市场标识（0=深市，1=沪市） |
| f14 | 名称 |
| f6 | 成交额 |
| f62 | 主力净流入（超大单+大单） |
| f184 | 主力净流入占比 |
| f66 | 超大单净流入 |
| f69 | 超大单净流入占比 |
| f72 | 大单净流入 |
| f75 | 大单净流入占比 |
| f78 | 中单净流入 |
| f81 | 中单净流入占比 |
| f84 | 小单净流入 |
| f87 | 小单净流入占比 |

### 应用场景
- **A股净流入排名**：`fid=f62&po=1`（按主力净流入降序）
- **涨幅排名**：`fid=f3&po=1`（按涨跌幅降序）
- **跌幅排名**：`fid=f3&po=0`（按涨跌幅升序）
- **成交额排名**：`fid=f6&po=1`（按成交额降序）

## 通用说明

### 错误处理
所有接口均返回 `rc` 或 `ErrCode` 字段表示请求状态：
- 0 表示成功
- 非0 表示失败，可通过 `ErrMsg` 查看错误信息

### 数据更新频率
1. 股票行情：实时更新（约1秒/次）
2. 资金流向：每15分钟更新
3. 基金净值：每日更新（T+1日）
4. 基金估值：盘中实时估算（约1分钟/次）

### 注意事项
1. 所有接口均存在跨域限制，生产环境建议通过后端代理
2. 高频访问可能导致IP被封，建议合理控制请求频率
3. 部分接口参数需要URL编码（如中文关键词）
4. 基金代码需区分市场（0=深市，1=沪市，101=国际）

### 示例代码（JavaScript）
```javascript
// 获取股票行情
async function getStockQuotes(codes) {
  const marketMap = { sz: 0, sh: 1 };
  const secids = codes.map(code => {
    const [prefix, num] = code.match(/([a-z]+)(\d+)/i).slice(1);
    return `${marketMap[prefix.toLowerCase()]}.${num}`;
  }).join(',');

  const res = await fetch(`https://push2.eastmoney.com/api/qt/ulist.np/get?fltt=2&fields=f2,f3,f4&secids=${secids}`);
  return await res.json();
}

// 获取基金信息
async function getFundInfo(fundCodes) {
  const res = await fetch(`https://fundmobapi.eastmoney.com/FundMNewApi/FundMNFInfo?Fcodes=${fundCodes.join(',')}`);
  return await res.json();
}
```
